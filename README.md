🏀  NBA 实时战报
nba_live.py  ·  使用说明
坐着看比赛，播报自动来
这是什么
一个 NBA 实时战报桌面工具，比赛期间自动抓取直播数据，以图文+语音的方式播报每一条战报。
你只需要打开程序、输入比赛 ID、点开始——不用盯着弹幕，不用刷滚屏，播报直接说给你听。

功能一览
	功能	说明
📊	实时记分板	顶部大字显示比分、节次，自动更新
📝	自动文字播报	每条战报实时滚入，最新内容始终在顶部
🔊	语音播报	微软神经网络语音朗读战报，解放眼睛
🎙️	音色切换	云扬/晓晓/云希/晓伊，菜单实时切换
🏅	球队 Logo	自动匹配队名显示球队图标
📈	球员数据面板	实时更新上场球员得分/篮板/助攻等全项数据
🔄	智能去重	多重去重机制，同一条播报不会重复出现
📦	补页抓取	逐页拉取、不跳过，确保每条信息不遗漏


安装依赖
Python 3.8+ 环境，运行前先安装以下包：

包名	安装命令	用途
requests	pip install requests	网络请求，抓取直播数据
edge-tts	pip install edge-tts	微软神经网络语音合成
pygame	pip install pygame	音频播放

💡 提示  以上三个包均为纯 Python，Windows/Mac/Linux 均可直接 pip 安装，无需额外系统依赖。

使用步骤
第一步：获取比赛 ID
打开 直播吧（zhibo8.cc）找到今日 NBA 比赛，进入比赛页面，URL 中的数字即为比赛 ID。
示例：  https://zhibo8.cc/nba/2025/0319/123456.htm  →  ID 为 123456

第二步：启动程序
1.双击运行 nba_live.py，或命令行执行 python nba_live.py
2.在顶部输入框填入比赛 ID
3.点击 ▶ 开始直播 按钮
4.（可选）点击 🔇 语音 OFF 开启语音，从音色菜单选择喜欢的声音

语音功能使用微软 Edge TTS 神经网络语音，音质自然接近真人，完全免费。

可选音色

常见问题
Q：语音只播了第一句，之后没声音？
原生 SAPI 引擎在 Windows 上存在 COM 线程限制。本程序已改用 edge-tts + pygame 方案，确保已安装这两个包。

Q：Logo 不显示？
•确认 icon/ 文件夹与 nba_live.py 在同一目录
•文件名必须是中文队名，例如 湖人.png，格式为 PNG
•程序启动后控制台会打印 [logo] 开头的失败提示，可据此排查

Q：有播报信息漏掉？
程序会逐页抓取 last_sid+1 到 cur_sid 之间的全部页面，理论上不会丢包。如仍有遗漏，可能是网络超时，程序会自动重试。

Q：出现重复播报？
程序有双重去重机制：① max_sid 未变时跳过请求；② 每条 item 用唯一 ID（或内容哈希）过滤。如仍重复，请提 issue 并附上终端日志。
